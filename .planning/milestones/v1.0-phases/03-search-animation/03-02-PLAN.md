  <action>
    Create `src/hooks/useAnimation.ts` implementing the rAF animation loop.

    **Hook signature:**
    ```typescript
    export function useAnimation(): {
      speed: number                              // 0.5–5.0, for SpeedPanel
      setSpeed: (v: number) => void
      startAnimation: (map: maplibregl.Map, route: RouteResult, graph: OsmGraph) => void
      cancelAnimation: () => void
    }
    ```

    **Implementation details:**

    State and refs:
    - `const [speed, setSpeed] = useState<number>(1.0)`
    - `const speedRef = useRef<number>(1.0)` — kept in sync: `useEffect(() => { speedRef.current = speed }, [speed])`
    - `const rafHandleRef = useRef<number | null>(null)`

    `cancelAnimation` (useCallback, stable):
    - If rafHandleRef.current !== null: cancelAnimationFrame(rafHandleRef.current), set to null
    - NOTE: does NOT clear frontier layers — the caller decides whether to clear (clearFrontierLayers is called by the Map wiring in App/MapView on route reset)

    `startAnimation` (useCallback depending on [cancelAnimation]):
    - Call cancelAnimation() first
    - Filter virtual nodes: `const history = filterHistory(route.searchHistory)`
    - `const total = history.length`; if total === 0 return early
    - `let cursor = 0`; `const visited: [number,number][] = []` — accumulated across frames
    - Define inner `frame()` function:
      - `const nodesPerFrame = computeNodesPerFrame(speedRef.current)`
      - `const batch = history.slice(cursor, cursor + nodesPerFrame)`; `cursor += nodesPerFrame`
      - For each id in batch: `const coord = graph.nodes.get(id); if (coord) visited.push(coord)`
      - `const frontierCoords = batch.map(id => graph.nodes.get(id)).filter((c): c is [number,number] => c !== undefined)`
      - Call `updateFrontierLayers(map, visited, frontierCoords)`
      - `const pathSlice = slicePath(route.path, cursor, total)`
      - Call `updateRouteLayer(map, pathSlice)`
      - If cursor < total: `rafHandleRef.current = requestAnimationFrame(frame)`
      - Else (done): `updateRouteLayer(map, route.path)` (ensure full path shown); `rafHandleRef.current = null`
    - `rafHandleRef.current = requestAnimationFrame(frame)`

    Import `filterHistory, slicePath, computeNodesPerFrame` from `../lib/animationUtils`.
    Import `updateFrontierLayers, updateRouteLayer` from `../lib/mapHelpers`.
    Import `RouteResult` from `../lib/router`.
    Import `OsmGraph` from `../lib/osmParser`.
    Import `maplibregl` from `maplibre-gl`.
    Import `useState, useRef, useCallback, useEffect` from `react`.
