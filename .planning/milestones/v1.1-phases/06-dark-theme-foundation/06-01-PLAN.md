<task type="auto">
  <name>Task 1: Load Space Grotesk and define color token system</name>
  <files>index.html, src/index.css</files>
  <action>
In index.html, add two `<link>` tags inside `<head>` before the existing `<link rel="icon">`:
1. `<link rel="preconnect" href="https://fonts.googleapis.com">`
2. `<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>`
3. `<link href="https://fonts.googleapis.com/css2?family=Space+Grotesk:wght@300;400;500;600;700&display=swap" rel="stylesheet">`

In src/index.css:
1. Add a `:root` block at the very top (before existing rules) defining these CSS custom properties:
   - `--color-bg-base: #0a0a0a` — full-screen background
   - `--color-bg-panel: #1a1a2e` — floating panel backgrounds
   - `--color-bg-overlay: rgba(10, 10, 30, 0.85)` — translucent overlay surfaces
   - `--color-accent: #4488ff` — primary accent (buttons, links, active indicators)
   - `--color-accent-active: #2255cc` — darker accent for active/pressed states
   - `--color-text: #e0e0f0` — default body text
   - `--color-text-secondary: #aabbff` — labels and secondary info
   - `--color-text-muted: #8899cc` — hints and tertiary text
   - `--color-error: #ff4466` — error states
   - `--color-border: #2e2e4a` — default border
   - `--color-border-subtle: rgba(255, 255, 255, 0.15)` — subtle overlay borders

2. Replace the `font-family` value on line 7 (the `body, button, input, select, textarea` rule) with:
   `'Space Grotesk', system-ui, sans-serif`
   Keep `system-ui, sans-serif` as fallback.
  </action>
