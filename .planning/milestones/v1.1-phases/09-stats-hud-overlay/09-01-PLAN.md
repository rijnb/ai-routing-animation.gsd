Create src/components/StatsHud.tsx as a renamed, visually overhauled replacement for StatsPanel.tsx.

**Props interface:** Keep identical prop names to StatsPanel — zero App.tsx prop changes needed:
```typescript
export interface StatsHudProps {
  nodesExplored: number
  totalNodes: number
  distanceKm: number | null
  travelTimeSeconds: number | null
  visible: boolean
}
```

**Visibility:** When `visible` is false, return null (same as current). Add a CSS opacity/transition fade-in: when visible, transition from opacity 0 to 1 over ~300ms. Use React state + useEffect to trigger the transition after mount-with-visible, or use a simple CSS transition on the container with opacity based on visible prop. Keep it lightweight — no animation library.

**Layout — top-level container:**
- `position: 'fixed'` (not absolute — HUD is viewport-pinned, independent of map container)
- `top: '16px', right: '16px'`
- `zIndex: 500` (above ControlPanel at 400)
- `background: '#1a1a2e'`
- `border: '1px solid #4488ff'`
- `borderRadius: '4px'`
- `padding: '10px 16px'`
- `display: 'grid'`, `gridTemplateColumns: 'repeat(3, auto)'`, `gap: '0 20px'`
- `fontFamily: "'Space Grotesk', sans-serif"`
- `transition: 'opacity 0.3s ease'`
- `opacity: visible ? 1 : 0` (combined with the return null guard: only render when visible, fade handled on initial render — or use a mounted/visible two-state approach if preferred)

**Three stat columns — each rendered as a `<div>` containing label + value:**

DIST column:
- Label: `"DIST"` in `#aabbff`, fontSize `10px`, fontWeight `600`, letterSpacing `0.1em`, textTransform `'uppercase'`, marginBottom `2px`
- Value: `distanceKm !== null ? distanceKm.toFixed(2) + ' km' : '—'` in `#e0e0f0`, fontSize `18px`, fontWeight `700`
- Note: `distanceKm` prop is already in km (StatsPanel uses `distanceKm.toFixed(2)` directly). Use the same format for consistency.

TIME column:
- Label: `"TIME"` — same label style as DIST
- Value: `travelTimeSeconds !== null ? formatTime(travelTimeSeconds) : '—'` in same value style

NODE column:
- Label: `"NODE"` — same label style
- Value: `{nodesExplored.toLocaleString()} / {totalNodes.toLocaleString()}` — same value style but fontSize `14px` (slightly smaller — the slash-separated pair is longer)

Import `formatTime` from `'../lib/routeStats'` (only `formatTime` is needed; distance formatted inline).

**Do not add:** Separator lines, header text, title label, or any decorative elements beyond the border. Keep it minimal — instrument readout, not a panel.
