<task type="auto">
  <name>Task 2: Wire useAnimation and SpeedPanel into App.tsx</name>
  <files>src/App.tsx</files>
  <action>
    Modify `src/App.tsx` to wire the animation:

    **1. Import useAnimation:**
    ```typescript
    import { useAnimation } from './hooks/useAnimation'
    import { SpeedPanel } from './components/SpeedPanel'
    import { clearFrontierLayers } from './lib/mapHelpers'
    ```

    **2. Call useAnimation hook:**
    ```typescript
    const { speed, setSpeed, startAnimation, cancelAnimation } = useAnimation()
    ```

    NOTE: `startAnimation` requires a `maplibregl.Map` reference. MapView owns the map internally. The cleanest approach (per RESEARCH.md recommendation) is to add an `onMapReady` callback prop to MapView that receives the map instance and stores it in a ref in App.tsx:
    ```typescript
    const mapRef = useRef<maplibregl.Map | null>(null)
    // In MapView.tsx: call onMapReady?.(map) after map.on('load') fires
    // In App.tsx: onMapReady={m => { mapRef.current = m }}
    ```

    Add `onMapReady?: (map: maplibregl.Map) => void` to MapViewProps and call `onMapReady?.(map)` inside the `map.on('load')` handler in MapView.tsx (AFTER loadedRef.current = true).

    **3. Auto-start animation when route changes:**
    ```typescript
    const prevRouteRef = useRef<RouteResult | null>(null)
    useEffect(() => {
      if (route && route !== prevRouteRef.current && graph && mapRef.current) {
        prevRouteRef.current = route
        startAnimation(mapRef.current, route, graph)
      }
    }, [route, graph, startAnimation])
    ```

    **4. Cancel animation on new map click + clear frontier layers:**
    Wrap `handleMapClick` so cancelAnimation is called first:
    ```typescript
    const handleMapClickWithCancel = useCallback((lngLat: [number,number]) => {
      cancelAnimation()
      if (mapRef.current) clearFrontierLayers(mapRef.current)
      handleMapClick(lngLat)
    }, [cancelAnimation, handleMapClick])
    ```
    Pass `handleMapClickWithCancel` to MapView's `onMapClick` prop instead of `handleMapClick`.

    **5. Clear animation state on OSM reset:**
    When stage returns to 'idle' (new file loaded or reset), cancel and clear:
    ```typescript
    useEffect(() => {
      if (stage === 'idle') {
        cancelAnimation()
        if (mapRef.current) clearFrontierLayers(mapRef.current)
      }
    }, [stage, cancelAnimation])
    ```

    **6. Render SpeedPanel:**
    Add `<SpeedPanel speed={speed} onSpeedChange={setSpeed} visible={route !== null} />` inside the map container div (sibling to `<MapView />`).

    **7. Pass graph and onMapReady to MapView:**
    ```tsx
    <MapView
      ...existing props...
      graph={graph}
      onMapReady={m => { mapRef.current = m }}
      onMapClick={handleMapClickWithCancel}
    />
    ```
